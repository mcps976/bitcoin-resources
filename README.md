# Bitcoin Learning Resources

[![License: CC0-1.0](https://img.shields.io/badge/License-CC0%201.0-lightgrey.svg)](http://creativecommons.org/publicdomain/zero/1.0/)

A comprehensive, curated collection of Bitcoin learning materials, development resources, and community tools.

## 📚 What's Inside

This repository contains a meticulously organized collection of:

- **Books & Papers** - Essential reading from beginner to advanced
- **Official Documentation** - Bitcoin Core, BIPs, and protocol specs
- **Video Courses** - Free and paid educational content
- **Development Resources** - Tools for building on Bitcoin
- **Explorers & Analytics** - Block explorers and on-chain analysis
- **News & Media** - Newsletters, podcasts, and trusted sources
- **Security & Privacy** - Best practices and tools
- **Lightning Network** - Layer 2 scaling resources
- **Wallets** - Reviewed wallet options for all use cases
- **Community** - Forums, discussion groups, and organizations

## 🎯 Who Is This For?

- **Beginners** - Start with the foundational books and courses
- **Developers** - Comprehensive development resources and APIs
- **Node Operators** - Guides for running Bitcoin and Lightning nodes
- **Researchers** - Academic papers and technical specifications
- **Privacy-Conscious Users** - Privacy tools and best practices
- **Anyone** interested in deepening their Bitcoin knowledge
